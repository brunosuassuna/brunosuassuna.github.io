Sobre Mim

Engenharia de dados com propósito e impacto

Engenheiro de Dados | The Store Intelligence

🎓 Sou graduado em Análise e Desenvolvimento de Sistemas e pós-graduado em Ciência de Dados e Analytics pela PUC-Rio. Minha jornada é pautada pela busca constante por soluções de dados que gerem valor real para os negócios.

💡 Acredito que os dados vão muito além de responder perguntas: eles revelam padrões, antecipam oportunidades e fundamentam decisões estratégicas com alto potencial de transformação.

🚀 Minha missão é projetar e implementar pipelines de dados eficientes, escaláveis e inteligentes, utilizando tecnologias como Azure Data Factory, Databricks, Power BI e SQL. Tudo isso com foco em performance, governança e impacto mensurável.

🔍 Estou sempre em busca de aprimorar processos, integrar dados de forma inteligente e tornar a informação acessível para orientar decisões com segurança e agilidade.
